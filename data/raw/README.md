# Dataset

This project uses the **DataCo Smart Supply Chain Dataset**.

## Download Instructions

Due to GitHub file size limitations, the dataset is not included in this repository.

### Option 1: Download from Kaggle
1. Visit: https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis
2. Download `DataCoSupplyChainDataset.csv`
3. Place in this directory (`data/raw/`)

### Option 2: Direct Download
[Alternative download link if available]

## Dataset Information

- **Size:** ~180,000 records
- **Format:** CSV
- **Encoding:** latin1
- **Columns:** 53 features including order details, shipping info, customer data

## Required File
```
data/raw/DataCoSupplyChainDataset.csv
```

After downloading, your structure should look like:
```
data/
└── raw/
    ├── README.md (this file)
    └── DataCoSupplyChainDataset.csv
```
